Relatório do Projeto

Introdução

O objetivo do projeto foi desenvolver um aplicativo mobile para auxiliar na alimentação e dieta das pessoas utilizando IA, proporcionando uma ferramenta acessível para orientação nutricional. O projeto esteve alinhado com o Objetivo de Desenvolvimento Sustentável (ODS) 3 – Saúde e Bem-Estar e utilizou React Native + Node JS como linguagem, pois buscou promover hábitos alimentares saudáveis e acessíveis a um público que enfrenta dificuldades para obter orientação profissional.

Interface

A interface do aplicativo foi projetada para ser intuitiva e acessível, permitindo que os usuários naveguem facilmente entre as funcionalidades principais.

Passo 1 - Coleta de Dados do Usuário

Nesta fase, o aplicativo solicita informações do usuário, como idade, peso, altura e objetivos nutricionais. Esses dados são essenciais para personalizar a experiência e fornecer recomendações adequadas.

Passo 2 - Geração do Plano Alimentar

Com base nas informações fornecidas, o aplicativo junto com a Inteligência Artificial (Gemini IA) gera um plano alimentar personalizado. Esse plano inclui sugestões de refeições balanceadas, respeitando as preferências do usuário.

Conclusão

O desenvolvimento do aplicativo trouxe resultados positivos, permitindo que os usuários tenham acesso a um suporte básico sobre alimentação saudável. O projeto teve impacto na conscientização alimentar dos participantes e demonstrou potencial para futuras melhorias. Foram identificadas áreas que podem ser aprimoradas, e futuras atualizações irão incorporar novas funcionalidades para tornar a experiência ainda mais eficiente.
